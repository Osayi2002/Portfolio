Premier League Teams Database

Overview

This project involves designing a database for Premier League teams to manage rankings, fanbases, and team details.

Key Features

  • Database with three tables: Teams, Rankings, and Fanbases.
  • Relationships managed using foreign keys for accurate data integrity.
  • Queries to calculate team performance and analyze fanbases.

Conclusion

The database helps analyze team rankings, performance metrics, and fan engagement, providing valuable insights into the Premier League.
