Sri Lanka Economic Analysis

Overview

This project explores economic data for Sri Lanka, focusing on GDP growth, inflation, public debt, and trade balances.

Key Features

  • Comparison of Sri Lanka's economic performance with global trade metrics and neighboring South Asian countries.
  • Visualizations of GDP growth, inflation trends, and trade balance over time.
  • Interactive data exploration using Python and IPython widgets.

Project Code Highlights

# Load Data
import pandas as pd
from IPython.display import display

# Data visualization
import matplotlib.pyplot as plt
import seaborn as sns

# Widgets for interactivity
from ipywidgets import interact

# Load Sri Lanka Economic Data from website
url = "https://www.focus-economics.com/countries/sri-lanka/"
tables = pd.read_html(url)
df = tables[0]

# Filter relevant indicators including Exports and Imports
indicators = [
    "Economic Growth (GDP, ann. var. %)", 
    "Inflation (CCPI, ann. var. %, aop)", 
    "Unemployment (% of active population, aop)", 
    "Public Debt (% of GDP)", 
    "Merchandise Trade Balance (USD bn)",
    "Exports (G&S, ann. var. %)",
    "Imports (G&S, ann. var. %)"
]

# Data cleaning and exploration
filtered_data = df[df['Unnamed: 0'].isin(indicators)]
filtered_data.columns = ['Indicator', '2019', '2020', '2021', '2022', '2023']
filtered_data.set_index('Indicator', inplace=True)
cleaned_data = filtered_data.transpose()

print("Sri Lanka Data:")
display(cleaned_data)

# Visualization Example
sns.lineplot(data=cleaned_data, x='Year', y='Economic Growth (GDP, ann. var. %)')
plt.title("Sri Lanka GDP Growth Over Time")
plt.show()

Conclusion

The analysis highlights challenges like inflation and trade deficits while showcasing how Sri Lanka's economic performance compares to regional and global trends. Recommendations include bolstering export efforts and implementing fiscal policies to address public debt and inflation.
