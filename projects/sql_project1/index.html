Event Management Database

Overview

This project involves creating a database to manage event details, including venues, event types, and ticket pricing.

Key Features

  • Normalized database design with three tables: Venues, Events, and Event Types.
  • Foreign key constraints to maintain relationships between entities.
  • Complex queries to calculate statistics and generate reports.

Conclusion

The database provides insights into event management, ticket pricing, and venue details, enhancing the efficiency of event handling.
